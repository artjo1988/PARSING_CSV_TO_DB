Описание. Данный приложения разработано c использавнием среды разработки Intellij IDEA, Maven, SpringBoot, PostgreSQL, Jdbc, log4j2. Приложение предназначено для копирования csv-файлов определённого представления, которые представлены в задании

1.Необходимо скопировать jar-файл себе на компьютер (с OS Windows) 
2.Создать в корне диска, где расположен jar-файл две папки source (папка для расположения исходных файлов) и processed (папка для располжения результирующих файлов), если папки не были созданы, приложение их создат, проинформирует об этом в консоли и завершит свою работу (с кодом ошибки -1), если csv-файлы не находились в папке  source на момент запуска приложения, приложение также завершит свою работу (с кодом ошибки -1) с информированием в консоли о необходимсоти подготовить все необходимые папки и файлы к работе.
3.При созданных необходимых папках и наличии csv-файлов в папке source, приложение скопирует исходный файл в папку processed, а исходный файл удалит, что будет говорить об успешном копировании в БД, если удаление успешно скопированного файла не произойдет, то приложение проинформирует об этом
